# Flujo De Trabajo

## Llegada De Mercancia

1. El dueno abre Productos.
2. Toma o sube foto.
3. Captura nombre, categoria, talla, color, costo, precio y cantidad.
4. El sistema calcula margen.
5. El producto queda disponible para venta.

## Venta En Mostrador

1. El dueno abre Nueva venta.
2. Busca producto por nombre, categoria o SKU.
3. Agrega producto al carrito.
4. Aplica descuento si hace falta.
5. Elige metodo de pago.
6. Finaliza venta.
7. El sistema descuenta inventario.
8. Opcionalmente guarda el cliente.

## Cambio O Devolucion

1. Busca venta.
2. Selecciona producto devuelto.
3. Define si vuelve a inventario.
4. Registra motivo.
5. El sistema ajusta stock y dinero.

## Cierre Del Dia

1. Abre Corte diario.
2. Revisa ventas totales.
3. Compara efectivo esperado contra efectivo real.
4. Revisa transferencias y tarjetas.
5. Ve productos mas vendidos.
6. Atiende alertas de stock bajo.

## Reposicion De Stock

1. Abre Inventario.
2. Filtra por stock bajo.
3. Decide que comprar.
4. Registra compra al recibir mercancia.
